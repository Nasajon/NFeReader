Bem vindo ao PrototipoMda

Essa é uma aplicação demo para demonstrar como uma aplicação gerada pelo MdaPhpTansformer pode ser utilizada e/ou modificada

Requisitos

  • PHP 5.6.30
  • NodeJs 6.11.0
  • Json-server https://github.com/typicode/json-server

Primeiros passos

  • Na raiz do projeto, execute o comando ./prepareEnvironment.sh para preparar o ambiente

Incluíndo um Novo Caso de Uso na Aplicação

[caminho do projeto] = Caminho absoluto da pasta do projeto (exemplo: /var/www/hatml/PrototipoMda).

  • Coloque o arquivo xml referente ao novo caso de uso na pasta src/xmls

Estendendo um Template

  • Coloque o template na pasta src/view

Incluindo arquivos .js

  • Coloque o template na pasta src/js
  • Inclua o path do arquivo webpack.config.js
  • Execute o comando na raiz node_modules/webpack/bin/webpack.js para atualizar o arquivo js utilizado pela aplicação incluindo as informações do arquivo .js recem adicionado

Configuração da aplicação

As configurações da aplicação encontram-se em app.yml
